## Palindrome Universe Game
A futuristic multiplayer word battle game where logic meets speed.
Challenge your friends in real-time or play offline duels in a visually immersive neon arena.

## Overview

Palindrome Universe is an interactive browser-based game designed to test your thinking speed and pattern recognition skills. Players compete by entering valid palindrome words under pressure. With both offline duel mode and online room-based multiplayer, the game delivers a competitive and engaging experience.

## What is a Palindrome?

A palindrome is a word that reads the same forward and backward.

Examples:

RADAR

LEVEL

KAYAK

ROTOR
